The [Montana Mesonet Dashboard](https://mesonet.climate.umt.edu/dash/) provides a suite of tools that assist with agricultural land management. They are available under the "Ag Tools" tab of the dashboard. Note that this toolset is only available on the desktop version of the dashboard (not mobile). The individual tools are listed below, with links to a more detailed overview of each tool:

1. [Reference Evapotranspiration](/mesonet/ag_tools/etr/)
2. ["Feels Like" Temperature](/mesonet/ag_tools/feels_like/)
3. [Crop Growing Degree Days](/mesonet/ag_tools/gdds/)
4. [Soil Profile Plot](/mesonet/ag_tools/soil_profile/)
5. [Livestock Risk Index](/mesonet/ag_tools/risk/)
6. [Soil Water Potential](/mesonet/ag_tools/swp/)
